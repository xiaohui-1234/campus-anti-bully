# 数据库设计

数据库名：`campus`。建表脚本位于：

```text
sql/campus.sql
campus-server/src/main/resources/sql/campus.sql
```

字符集使用 `utf8mb4`。

## device

设备基础表。

| 字段 | 说明 |
| --- | --- |
| `id` | 自增主键 |
| `device_id` | 对外设备唯一标识 |
| `product_type` | 产品类型，如 `anti_bullying` |
| `device_secret` | 设备密钥哈希，禁止保存明文 |
| `create_time` | 注册时间 |
| `last_online_time` | 最后在线时间 |

索引：

- `uk_device_id(device_id)`
- `idx_product_type(product_type)`

## user

用户表。

| 字段 | 说明 |
| --- | --- |
| `id` | 自增主键 |
| `user_id` | 对外用户唯一标识 |
| `openid_hash` | 微信 openid 哈希，禁止保存明文 openid |
| `nickname` | 昵称 |
| `email` | 邮箱 |
| `phone` | 手机号 |
| `avatar_url` | 头像 URL |
| `role` | `USER` 或 `ADMIN` |
| `create_time` | 注册时间 |

索引：

- `uk_user_id(user_id)`
- `uk_openid_hash(openid_hash)`
- `idx_phone(phone)`
- `idx_email(email)`

## user_device_bind

用户和设备绑定关系表。用户自定义设备名称、位置和备注存储在绑定关系上，不写回设备基础表。

| 字段 | 说明 |
| --- | --- |
| `id` | 自增主键 |
| `device_table_id` | 外键，关联 `device.id` |
| `user_table_id` | 外键，关联 `user.id` |
| `bind_time` | 绑定时间 |
| `device_name` | 用户自定义设备名 |
| `location` | 设备位置 |
| `note` | 备注 |

约束：

- `uk_user_device(user_table_id, device_table_id)`
- 外键 `fk_bind_device`
- 外键 `fk_bind_user`

## event

报警事件表。

| 字段 | 说明 |
| --- | --- |
| `id` | 自增主键 |
| `mqtt_msg_id` | MQTT 消息 ID，用于数据库兜底去重 |
| `device_table_id` | 事件所属设备，关联 `device.id` |
| `event_id` | 对外事件 ID |
| `event_type` | 事件类型，如 `VOICE`、`BUTTON` |
| `alarm_info` | 报警描述，如 `fight`、`button_alarm` |
| `file_key` | MinIO 对象 Key |
| `file_status` | `UPLOADING`、`SUCCESS`、`FAILED` |
| `push_status` | `PENDING`、`PUSHED`、`FAILED` |
| `read_status` | `UNREAD`、`READ` |
| `event_time` | 事件发生时间 |

索引：

- `uk_event_id(event_id)`
- `uk_device_mqtt_msg(device_table_id, mqtt_msg_id)`
- `idx_device_event_time(device_table_id, event_time)`
- `idx_push_status(push_status)`
- `idx_read_status(read_status)`

## device_config_wifi

设备 WiFi 配置列表，只保存 WiFi 名称，不保存 WiFi 密码。

| 字段 | 说明 |
| --- | --- |
| `id` | 自增主键 |
| `device_table_id` | 外键，关联 `device.id` |
| `config_id` | 设备内部配置编号 |
| `wifi_name` | WiFi 名称 |
| `set_time` | 更新时间 |

约束：

- `uk_device_config(device_table_id, config_id)`
- 外键 `fk_wifi_device`

## 开发种子数据

根目录 `sql/campus.sql` 包含：

- 开发设备：`dev001`
- 开发管理员：`usr_admin`

生产环境应替换或移除种子数据，且不得使用示例密钥。
