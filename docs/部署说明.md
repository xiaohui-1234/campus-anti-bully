# 部署说明

## 依赖

- JDK 17+
- Maven 3.9+
- MySQL 8
- Redis
- MinIO
- EMQX
- Node.js 20+

## 后端

```bash
cd campus-server
mvn clean package
java -jar target/campus-server-1.0.0.jar
```

生产环境需要替换：

- `campus.security.jwt.secret`
- `spring.datasource.password`
- `campus.wx.miniapp.app-id`
- `campus.wx.miniapp.app-secret`
- `campus.minio.secret-key`
- `campus.mqtt.password`

## 后台

```bash
cd campus-admin-web
npm install
npm run build
```

构建产物在 `dist`。

## 小程序

使用微信开发者工具打开 `campus-miniprogram`，并把 `config/env.js` 改为实际 API 与 WebSocket 地址。
